Новшества EcmaScript 6

В этом документе загружается и выполняется Javascript код, демонcтрирующий особенности этого языка программирования, появившиеся в новом стандарте EcmaScript 6.
Основной вывод результатов работы скрипта осуществляется в консоль браузера.

Код разделён на две части. Первая загружается из файла js-begin-ES6.js.

Вторая часть кода содержит синтаксические конструкции, приводящие к ошибке на старых версиях браузеров, имеющих только частичную поддержку ES6 (в результате чего скрипт не запускается). Поэтому эта вторая часть кода помещена в файл js-begin-ES6-extended.js, который подключается на странице только в случае наличия в браузере необходимого функционала.
